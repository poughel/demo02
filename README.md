## 🎯 项目特点

### DataSophon 是什么？
DataSophon（数据智子）是一个面向大数据平台的自动化部署、运维、监控和管理系统，致力于：
- 快速部署大数据集群（支持300+节点）
- 统一管理多种大数据组件（HDFS、YARN、Hive、HBase、Spark等）
- 全面监控集群健康状态
- 智能告警和自动化运维
- 支持国产化（ARM架构、国产操作系统）

### 技术亮点
- **分层架构**：清晰的模块划分，易于维护和扩展
- **Actor模型**：使用Akka实现高性能异步消息处理
- **DDD设计**：领域驱动设计，业务逻辑清晰
- **模板化配置**：FreeMarker模板，灵活生成配置文件
- **前后端分离**：Vue.js + Spring Boot
